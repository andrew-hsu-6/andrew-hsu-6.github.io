## üé® Customization Guide

### Personal Information

#### Update Contact Details
In `index.html`, find and update these sections:

```html
<!-- Update email -->
<p>your.email@columbia.edu</p>

<!-- Update LinkedIn -->
<p>linkedin.com/in/yourprofile</p>

<!-- Update GitHub -->
<p>github.com/yourusername</p>
```

#### Update About Section
Modify the about section to reflect your specific background:

```html
<p>
    I'm a dedicated Computer Science student at Columbia University with a strong foundation 
    in programming fundamentals and algorithmic thinking...
</p>
```

### Projects Section

Replace the example projects with your actual work:

```html
<div class="project-card">
    <div class="project-content">
        <h3>Your Project Name</h3>
        <p>Description of your project...</p>
        <div class="project-tech">
            <span class="tech-tag">Java</span>
            <span class="tech-tag">Spring Boot</span>
        </div>
        <div class="project-links">
            <a href="https://github.com/yourusername/project" class="project-link">
                <i class="fab fa-github"></i> Code
            </a>
            <a href="https://yourproject.com" class="project-link">
                <i class="fas fa-external-link-alt"></i> Live Demo
            </a>
        </div>
    </div>
</div>
```

### Skills Section

Update your technical skills and proficiency levels:

```html
<div class="skill-item">
    <div class="skill-icon">
        <i class="fab fa-java"></i>
    </div>
    <div class="skill-info">
        <span class="skill-name">Java</span>
        <div class="skill-level">
            <div class="skill-bar" data-level="90"></div>
        </div>
    </div>
</div>
```

**Skill Levels**: Change `data-level` from 0-100 to reflect your proficiency.

### Education Section

Update your academic information:

```html
<div class="education-item">
    <div class="education-details">
        <h4>Columbia University</h4>
        <p>Bachelor of Science in Computer Science</p>
        <span class="education-period">2023 - Present</span>
    </div>
</div>
```

## üåê Deployment Options

### Free Hosting Options:

1. **GitHub Pages** (Recommended)
   - Push your code to a GitHub repository
   - Enable GitHub Pages in repository settings
   - Your site will be available at `username.github.io/repository-name`

2. **Netlify**
   - Drag and drop your project folder to netlify.com
   - Get instant deployment with custom domain options

3. **Vercel**
   - Connect your GitHub repository
   - Automatic deployments on code changes

### Custom Domain:
- Purchase a domain (e.g., `yourname.com`)
- Point it to your hosting service
- Update the domain in your hosting settings
